# Odisents Player

Un reproductor multimedia moderno para Windows.

## Descarga

Ve a la seccion de [Releases](https://github.com/zlukiniDeveloper/odisents-player-releases/releases) para descargar la ultima version.

## Caracteristicas

- Reproductor multimedia con MPV
- Integracion con TMDB para metadatos de peliculas y series
- Subtitulos automaticos via OpenSubtitles
- Sincronizacion con Trakt
- Integracion con Radarr/Sonarr

## Requisitos

- Windows 10/11
- Claves de API (configurables desde la app):
  - TMDB (requerido)
  - OpenSubtitles (opcional)
  - Trakt (opcional)
